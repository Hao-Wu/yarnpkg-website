### macOS

如果你还没有安装 [Node.js](https://nodejs.org/)，那么你需要首先 [安装 Node.js](https://nodejs.org/)。

#### Homebrew

你可以通过 [Homebrew 包管理工具](http://brew.sh/) 安装Yarn，当你未安装 Node.js 时，这个操作将自动为你安装 Node.js。

```sh
brew update
brew install yarn
```
